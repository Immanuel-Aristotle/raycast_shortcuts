# Raycast Shortcuts

Standalone Raycast script commands grouped by task type.

## Layout

| Directory | Scripts |
| --- | --- |
| `clipboard/` | Clipboard-focused converters and text cleanup helpers. |
| `files/` | File routing, folder opening, and file share-link helpers. |
| `location/` | Location lookup and coordinate conversion helpers. |
| `media/` | Image conversion and media metadata calculation helpers. |
| `messages/` | Message parsing helpers. |
| `quicklinks/` | External service quicklink helpers. |
| `text/` | Subtitle and plain-text conversion helpers. |
